3DViewer v2.0-1 Documentation

3DViewer - application for viewing wireframe models from an .obj files

Installing

To install the application run make install command

Main features

  1. Loading a wireframe model from an .obj file (vertices and surfaces list support only)
  2. Translating the model by a given distance in relation to the X, Y, Z axes with sliders
  3. Rotating the model by a given angle relative to its X, Y, Z axes with sliders
  4. Scaling the model by a given value with slider
  5. Settings for type, color and thickness of edges
  6. Settings for display method, color and size of vertices
  7. Choosable background color
  8. Apllied settings saving between restarts
  9. Saving captured images in jpeg, bmp formats
  10. Recording small screencasts into gif-animation (640x480, 10fps, 5s)

User Interface

Consists of:

  • Visualisation area for the wireframe model
  • Sliders for affine transformations
  • Buttons for saving images and gif-files
  • Information about loaded model (filename, number of edges and vertices)
  • Button for choosing a model
